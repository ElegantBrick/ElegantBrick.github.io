Node-sass 安装不成功

2016-12-19

安装node-sass不成功
提示错误
Downloading binary from https://github.com/sass/node-sass/releases/download/v3.13.1/win32-x64-51_binding.node
Cannot download “https://github.com/sass/node-sass/releases/download/v3.13.1/win32-x64-51_binding.node“:




诸如此类错误，据说是因为亚马逊代理什么的，网络连接不成功。

解决方案

1.先根据报错的网站把不能安装的文件手动下载下来，放在本地一个文件夹下面，最好路径没有空格没有汉字。

比如e:根目录（e:/win32-x63-51_binding.node）。

2.命令行输入 set SASS_BINARY_PATH=E:/win32-x64-46_binding.node 之后，再试一次安装。

3.如果出现如下错误：

gyp verb check python checking for Python executable "python2" in the PATH.

a.下载对应版本的Python.

b.安装pathon,并配置环境变量。

c.然后退出命令行，再进入命令行，输入 python.安装并成功配置环境变量则会输出python版本。

4.进入到项目中，在命令行中输入 set SASS_BINARY_PATH=E:/win32-x64-46_binding.node

最后，输入安装命令。

以上只是我这样解决这个问题的方式。

还有一种方式就是，直接本地删除 node_modules 文件夹。再npm install.
